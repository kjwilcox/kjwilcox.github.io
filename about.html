Computers and Programming

Hex

An implementation of the classic strategy game Hex, invented in 1942 by Piet Hein.

I implemented this game in college so that I could get my friends interested in this strategy game. I implemented network play, but I hadn't discovered source control at that point and ended up losing both the server-side code and the client-side networking code. This version contains only hotseat multiplayer.

Technologies: Python, SDL, TCP
