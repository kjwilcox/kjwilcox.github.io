Computers and Programming

I have loved computers from a very early age.

I taught myself to program at age 16 on a TI-83 graphing calculator. Ever since then, I have been learning and using new programming languages. I would consider myself a programming language polyglot. I love opportunities to learn new tools and langauges.

Languages and formats: Python, C, C++, Java, C#, PHP, Javascript, Basic, SQL, HTML/CSS, SVG, XML, JSON

Environments and protocols: Windows, Linux, Apache, IP, TCP, UDP, HTTP, REST, DNS

Tools: Make, Ant, Visual Studio, IzPack, NSIS, Bash
